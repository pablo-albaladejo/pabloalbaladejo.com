# SEO/GEO baseline — pabloalbaladejo.com — 2026-07-22

Snapshot from the three-agent audit (live technical + SERP/entity + repo).
Every future `reports/seo/timeseries/` entry is compared against this t0.

## Headline

- **On-page/technical: A-grade.** All 8 pages prerendered with unique
  titles/descriptions, self-canonicals, Person/ProfilePage/Article/FAQPage
  JSON-LD, llms.txt, RSS, build-time OG cards, no cloaking (byte-identical
  HTML for Googlebot/GPTBot/ClaudeBot/PerplexityBot), TTFB 31–38 ms,
  brotli+HTTP/2, real 404s, security headers.
- **Discovery is the failure: 0 pages indexed** on Google AND Bing
  (`site:` empty on both). The only indexed owned artifact is the GitHub
  source repo. Cause: site is weeks old, 0 backlinks, GSC "Verify" click
  pending (DNS token live), Bing unregistered.
- **The entity is fragmented.** ~8 namesakes contest the name (journalist
  @AlbaladejoPress owns the X real estate; historians/athletes/engineers);
  no knowledge panel; TWO LinkedIn profiles split the signal; dev.to bio is
  empty with no backlink (his best-ranking property); "deepgent founder"
  returns DeepSeek on every engine — the relation is stated nowhere.
- **His content ranks — on other people's domains.** The dev.to series
  "Observable AI Streaming on AWS" ranks #1/#2/#4 for its (distinctive,
  branded) title; on-site articles with generic titles are invisible.
- **Measurement was zero**: Umami env-gated (unconfirmed live), no
  CloudFront logs, no GSC/Bing data pipe.
- deepgent.com (surfaced once in SERPs) is a **parked drop-catch** (JulyDNS,
  registered 2025-07-05, serves nothing) — not a competing surface; canonical
  stays deepgent.net.
